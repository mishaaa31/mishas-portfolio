# Misha Yadav — Personal Portfolio

A modern, dark-mode personal portfolio website built with React, Vite, and Tailwind CSS.

## Tech Stack
- **React 18** + **TypeScript**
- **Vite** — lightning-fast build tool
- **Tailwind CSS** — utility-first styling
- **Framer Motion** — smooth animations
- **shadcn/ui** — accessible component library
- **Lucide React** — icons

## Features
- Single-page application with smooth scroll navigation
- Dark-mode-first with electric purple neon accents
- Glassmorphism project cards with hover animations
- Fully responsive (mobile hamburger nav)
- Sections: Hero · About · Experience · Projects · Content · Contact

## Local Development
```bash
pnpm install
pnpm --filter @workspace/portfolio run dev
```
